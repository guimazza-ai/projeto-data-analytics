## 📊 Projeto de Data Analytics com Power BI

Este projeto tem como objetivo desenvolver um **relatório analítico em Power BI**, aplicando conceitos de **análise de dados, criação de medidas e design orientado ao usuário**, a partir de uma base de dados de vendas.

O foco principal está em transformar dados brutos em **informações relevantes para tomada de decisão**, garantindo clareza, interatividade e boa experiência de consumo do relatório.

---

## 🎯 Objetivos do Projeto

- Criar uma **página de detalhes**, conforme proposto no desafio
- Planejar a disposição dos visuais considerando a **experiência do cliente**
- Aplicar boas práticas de **organização visual e storytelling com dados**
- Desenvolver as **medidas necessárias** para suportar as análises
- Permitir flexibilidade na estrutura do relatório (até **duas páginas**)

---

## 📐 Estrutura do Relatório

- O relatório pode ser composto por **uma ou duas páginas**, dependendo da disposição dos visuais
- Cada página deve apresentar informações claras e complementares
- Os visuais são organizados de forma a facilitar a interpretação dos dados e a navegação

---

## 📈 Visuais Utilizados no Relatório

O relatório pode conter os seguintes tipos de visuais:

- **TOP 3 Produtos**
  - Identificação dos produtos com melhor desempenho (vendas, lucro ou outro indicador)

- **Principais Países**
  - Destaque dos países com maiores valores de:
    - Vendas
    - Profit (ou outra métrica relevante)

- **Visuais de Agrupamento de Dados**
  - Consolidação de informações por categorias, segmentos ou períodos

- **Visuais de Compartimentação**
  - Análises segmentadas que permitem filtrar e comparar diferentes perspectivas dos dados

---

## 🧮 Medidas e Análises

- Criação de **medidas DAX** para cálculo de indicadores como vendas, lucro, médias e variações
- As medidas suportam os visuais e garantem consistência nas análises exibidas


Este projeto consolida competências essenciais em **Data Analytics com Power BI**, unindo análise, métricas e design orientado ao usuário.
